# roBa
roBaは[keyball](https://github.com/Yowkees/keyball/)に影響を受けたワイヤレスキーボードです  

特徴:
+ bluetooth(BLE)対応
+ 分割カラムスタッガード配列(キー数:42)
+ トラックボール搭載
+ 水平ロータリーエンコーダ(EVQWGD001)搭載
